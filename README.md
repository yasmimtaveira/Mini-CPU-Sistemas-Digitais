# Mini-CPU-Sistemas-Digitais 🖥
Projeto desenvolvido na cadeira de Sistemas Digitais.
- Linguagem de descrição de hardware: Verilog;
- Placa utilizada: FPGA DE2-115.

  Durante o projeto, pudemos praticar conceitos aprendidos durante as aulas e adquirir conhecimentos práticos, como a configuração do display lcd, tratamento de sinal dos números para os cálculos realizados pela ULA, escrita e leitura numa memória digital, entre outros.
